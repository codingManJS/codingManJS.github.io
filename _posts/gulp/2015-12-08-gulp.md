##  gulp-bump
>  看名字很糊涂。。其实是一个更新版本的插件

``` javascript

  gulp.task('bump', function(){
    return gulp.src('./package.json')
      .pipe(bump({type: 'patch'}))
      .pipe(gulp.dest('./'))
  })
  //bump(opts) 这个函数中是可以传入参数的
  //1.version 比如说{version: 1.0.1}
  //2.type  这其中有4个
  //prerelease：1.2.3-0
  //patch：1.2.4
  //minor：1.3.0
  //major：2.0.0
  //3.indent改变原来的json文件的缩进{indent: 4}
  //也可以在src中传入多个文件比如 bower.json

```
